# 📄 PDF Summarizer using RAG + Multi-Agent System

An AI-powered PDF summarizer that uses Retrieval-Augmented Generation (RAG) and a multi-agent pipeline to generate accurate and concise summaries from documents.

## 🚀 Features

- 📄 Extracts text from PDF files
- ✂️ Intelligent text chunking
- 🔢 Embedding generation
- 📦 FAISS vector database for similarity search
- 🔍 Retrieval-Augmented Generation (RAG)
- 🤖 Multi-agent pipeline (Analyzer, Summarizer, Editor)
- 🧠 Uses Gemini AI for text generation
- 🛡️ Fallback handling for API failures
- 📑 Clean formatted output

---

## 🧠 How It Works
DF → Text → Chunking → Embeddings → FAISS → Retrieval → AI Agents → Summary
